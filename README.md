# ThaqalaynTafsirData

Static JSON data for Shia Quranic tafsir (exegesis), served as the tafsir data
backend for [Thaqalayn](https://thaqalayn.netlify.app).

Deployed to: https://thaqalayntafsir.netlify.app

## Structure

```
editions.json           # Index of all tafsir editions
{edition_id}/           # e.g. ar.mizan, fa.nemooneh, en.noor
  1.json                # Surah 1 (Al-Fatihah)
  2.json                # Surah 2 (Al-Baqarah)
  ...
  114.json              # Surah 114 (An-Nas)
```

URLs:
- https://thaqalayntafsir.netlify.app/editions.json
- https://thaqalayntafsir.netlify.app/ar.mizan/1.json

## Data format

### `editions.json`

Array of edition metadata:

```json
[
  {
    "id": "ar.mizan",
    "name": "الميزان في تفسير القرآن",
    "name_en": "Al-Mizan fi Tafsir al-Quran",
    "author": "العلامة الطباطبائي",
    "author_en": "Allamah Tabatabai",
    "language": "ar",
    "source": "app-furqan"
  }
]
```

### Per-surah files (`tafsir/{edition_id}/{surah}.json`)

Block-reference format to dedupe commentary that spans multiple ayahs:

```json
{
  "edition": "ar.mizan",
  "surah": 1,
  "blocks": [
    "<div class=\"tafsir-arabic\">...</div>",
    "<div class=\"tafsir-arabic\">...</div>"
  ],
  "ayahs": [
    {"ayah": 1, "block": 0},
    {"ayah": 2, "block": 0},
    ...
    {"ayah": 6, "block": 1},
    {"ayah": 7, "block": 1}
  ]
}
```

## Sources

- **app-furqan**: Derived from SQLite databases at
  [github.com/app-furqan/quran-app-data](https://github.com/app-furqan/quran-app-data)
  (CC BY-ND 4.0). Provides al-Mizan, Nemooneh, Noor, and as-Safi in
  Arabic/Farsi/English/Urdu.
- **altafsir.com**: Classical Shia tafsirs scraped with permission (Royal Aal
  al-Bayt Institute for Islamic Thought, academic use). Provides al-Qummi,
  al-Tibyan, Majma' al-Bayan, al-Burhan, Bayan al-Saadah, Sadr al-Muta'allihin.

## Generation

Data is generated by scripts in the sibling
[`ThaqalaynDataGenerator`](https://github.com/narmafraz/ThaqalaynDataGenerator)
repo:

```bash
cd ThaqalaynDataGenerator
DESTINATION_DIR=../ThaqalaynTafsirData/ SOURCE_DATA_DIR=../ThaqalaynDataSources/ \
  py app/tafsir_converter.py      # app-furqan SQLite → JSON
DESTINATION_DIR=../ThaqalaynTafsirData/ SOURCE_DATA_DIR=../ThaqalaynDataSources/ \
  py app/altafsir_converter.py    # altafsir scraped data → JSON
```
